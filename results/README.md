# Results

- paper_reference records the values printed in the submitted manuscript.
- generated is overwritten by the reproduction scripts.

Always inspect `generated/run_metadata.json` before interpreting generated CSVs.
It records whether the committed artifacts came from a full run or the short
integration smoke test. Smoke-test numbers are not paper results.

The original multi-controller harness was not present in the available project
materials. Generated benchmark values are therefore not silently presented as
bit-for-bit reproductions of the archived manuscript values. The QLBS matrices,
profiles, variants, and prediction protocols are explicit and reproducible.
